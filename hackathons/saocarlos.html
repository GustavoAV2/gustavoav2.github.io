Solução apresentada

Aplicação faseada de RPA para Simplificação de tarefas

Desenvolvemos uma aplicação faseada para simplificar os processos internos da BestCenter, a empresa patrocinadora do Hackathon. A solução possui três blocos principais:

  • Formulário dinâmico
  • Orquestrador de E-mails
  • Analisador de crédito

Formulario Dinâmico

Otimização exibição de campos para usuário, com campos mais acertivos coletando apenas os dados necessários que preencham a documentação necessárioa conforme o tipo de cliente

Analisador de crédito

O analisador de crédito tem a função de centralizar as informação das fontes disponiveis, analisar os dados e com base nisso gerar uma nivel de risco, dentro do hackathon simulamos respostas da API do Serasa e da Receita Federal para desenvolver a lógica de analise de crédito do Cliente
